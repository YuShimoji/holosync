# SP-008: タイムスタンプ抽出・ジャンプ機能

## 概要

YouTube動画の説明文からタイムスタンプ（例: `1:23`, `01:23:45`）を検出し、クリックで該当位置にシークする。

## 対応フォーマット

- `M:SS` (例: `0:00`, `3:45`)
- `MM:SS` (例: `12:30`)
- `H:MM:SS` (例: `1:23:45`)
- `HH:MM:SS` (例: `01:23:45`)

行頭または空白の後にタイムスタンプが出現するパターンをチャプターマーカーとして扱う。

## 変更内容

### 1. player.js: タイムスタンプ解析とリンク化

`fetchVideoDescription()` 内で、説明文をプレーンテキストではなく、
タイムスタンプ部分を `<span class="ts-link" data-seconds="N">` でラップしたHTMLとして挿入する。

新規関数:
- `parseTimestamp(str)` → 秒数を返す（無効なら-1）
- `renderDescriptionWithTimestamps(desc, videoId)` → タイムスタンプをクリッカブルにしたDOM要素を返す

### 2. player.js: クリックハンドラ

`ts-link` クリック時に `sendCommand(iframe, 'seekTo', [seconds])` を実行。
イベント委譲で `info-description` 要素にリスナーを1つ追加。

### 3. CSS: タイムスタンプリンクのスタイル

`ts-link` にアクセントカラー、下線、hover時の背景を追加。

## 完了条件

- [ ] 説明文中のタイムスタンプ（M:SS, MM:SS, H:MM:SS）がクリッカブルなリンクとして表示される
- [ ] クリックで該当秒数にシークする
- [ ] タイムスタンプがない説明文は従来通りプレーンテキスト表示
- [ ] XSS対策: 説明文のHTML文字はエスケープされる
- [ ] lint通過
